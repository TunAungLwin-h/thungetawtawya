ThuNgeTaw Tawya Examination Management System
Project Overview

The ThuNgeTaw Tawya Examination Management System is a web-based application designed to manage religious examination registration, candidate information, and results digitally. It helps administrators organize student data, assign levels, generate roll numbers, and track examination performance in a fast and accurate way.

Features

Candidate registration and profile management

Automatic roll number generation

Multiple examination level support

Search and filter candidates easily

Record and publish examination results

Secure admin login with session protection

Structured MySQL database storage
